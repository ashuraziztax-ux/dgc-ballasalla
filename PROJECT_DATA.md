## Naming Convention

| Code | Meaning |
|---|---|
| **S** | Surface Water chamber/sewer |
| **F** | Foul Water chamber/sewer |
| **Number after letter** | Road/zone section (e.g. 4 = Section 4 road area) |
| **Decimal number** | Sequence along the run (e.g. S4.1 → S4.2 → S4.3...) |
| **Existing** prefix | Pre-existing infrastructure to connect into |
